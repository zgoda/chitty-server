Chitty chat
===========

Chitty chat is an educational project that builds simple WebSocket-powered chat. It consists of chat server, web client application and mobile client application. I expect to boost my proficiency in various aspects.

Educational aspects
-------------------

Server
^^^^^^

* asynchronous programming using `Trio <https://github.com/python-trio/trio>`_ library

Web client
^^^^^^^^^^

* more `Preact <https://preactjs.com/>`_
* TypeScript checks for regular JavaScript
* application state management with hooks only

Mobile client
^^^^^^^^^^^^^

* more `Flutter <https://flutter.dev/>`_
